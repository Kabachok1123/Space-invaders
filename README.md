# Space Invaders

Учебный проект на Python.

## Запуск

```powershell
python main.py
```

## Текущий этап

- создано окно с GUI
- добавлен простой бекграунд
- добавлен корабль игрока
- корабль двигается влево и вправо
- добавлены выстрелы игрока
- добавлены статичные враги
- враги исчезают при попадании
- добавлен счетчик очков

## Управление

- `A` или `Left` - движение влево
- `D` или `Right` - движение вправо
- `Space` - выстрел
- `Esc` - выход
